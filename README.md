# Wendy's Data Extraction Using XPath

## 📖 Overview

This project is a Python-based web scraping solution developed to extract Wendy's restaurant information using XPath. The scraper collects structured location data and demonstrates practical experience in XPath-based data extraction, HTML parsing, and web scraping techniques.

The project focuses on accurate data extraction and clean Python development practices.

---

## 🚀 Features

- XPath-based data extraction
- Automated data collection
- Structured data processing
- JSON output generation
- Error handling and logging
- Clean and maintainable codebase

---

## 🛠️ Technologies Used

- Python
- Requests
- XPath
- lxml
- JSON
- Logging

---

## 📊 Extracted Data

- Store Name
- Store URL
- Address
- City
- State
- Postal Code
- Contact Information
- Location Details
